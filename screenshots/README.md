## Screenshot Guidelines

### Setup Screenshots
- Show clear terminal/command line outputs
- Highlight important configuration steps
- Show .env file setup (with credentials redacted)

### Automation Process Screenshots
- Capture each major step of the automation
- Show browser window with LinkedIn interface
- Include terminal logs showing progress
- Highlight successful completion messages

### Streamlit App Screenshots
- Show the web interface in different states
- Capture form inputs and buttons
- Show success/error messages
- Include status indicators

## Adding Screenshots

1. Take screenshots at appropriate stages
2. Name them according to the convention above
3. Ensure sensitive information (passwords, API keys) is redacted
4. Use descriptive filenames that indicate the process step
